About

Hello!

I’m Ashley. I’ve been fascinated with the web since getting my first dial-up modem in middle school. Shortly thereafer, I decided that teaching myself HTML would be a cool thing to do, so I signed up for a Geocities account and started peeking at source code. I’ve been hooked ever since.

“The only way to do great work is to love what you do.”

—Steve Jobs

Nowadays I classify myself as a Front-End/User Interface Web Developer. That essentially means that I spend a lot of time making websites functional, interactive, and usable. I love what I do.

I’m passionate about responsive web development, user interface design, usability, and accessibility.

When I’m not building websites, you can probably find me playing old-school SNES games, crocheting, reading, cooking, or pretending to know something about renovating old houses.
